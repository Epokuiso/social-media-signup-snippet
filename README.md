## Description
This snippet has the intend to show how to use different social media accounts in order to signup a user into a system using **NestJS**.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
